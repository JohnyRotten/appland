# Вёрстка от [Htmlacademy](https://htmlacademy.ru/)

## Зависимости

* [MeteorJS](https://meteor.com)

## Запуск

* `git clone git@github.com:JohnyRotten/appland.git`
* `cd appland`
* `meteor`
* Открыть браузер по адресу [http://localhost:3000](http://localhost:3000)
